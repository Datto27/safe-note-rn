# Safe Note
To Save Your Information Safely

## Install libraries
```bash
/> yarn install
/> cd ios && pod install
```

## Start application
- yarn start

#### Run on android
- yarn run android

#### Run on ios
- yarn run ios

## Visit releases page to download specific build for your android
https://github.com/Datto27/safe-note-rn/releases
